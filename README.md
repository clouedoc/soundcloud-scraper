# soundcloud-scraper

A soundcloud scraper

## Run

`sax`

## Link binary

```bash
yarn link

## Make sure you have yarn bin added to path
yarn global bin # <- add "export PATH=$PATH:$(yarn global bin)" to your zshrc
```
